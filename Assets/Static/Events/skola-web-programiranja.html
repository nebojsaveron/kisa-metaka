Škola web programiranja

za iskusnjare od 12.5. do 2.6.

Za kraj ove akademske godine, predstavljamo vam novi ciklus Škola web programiranja. Uz želju za učenjem, mnogo entuzijazma i malo slobodnog vremena, nadamo se uvući vas u svijet web programiranja. Prateći sva 4 predavanja, steći ćete dovoljno znanja za kreiranje jednostavnih web stranica. Počet ćemo s osnovnim konceptima Interneta, HTML-om i CSS-om, preko ASP.NET-a i MVC-a te na kraju slijedi izrada vlastitog malog projekta. Predavanja će držati članovi DUMP-a, većinom i sami studenti FESB-a, na svoj – DUMP način. Kao i do sada, predavanja će se održavati na FESB-u, subotom u 17 sati. Nakon subotnjeg predavanja, održat ćemo i nedjeljnu radionicu na računalima, kako biste mogli primijeniti naučeno. Uz opuštenu atmosferu i već poznate predavače, sigurno ćete s oduševljenjem pratiti predavanja i uploviti u svijet web programiranja.

Predavanja u sklopu ovog ciklusa

  • 1. Internet, HTML i CSS

    Marko Matijević, Davor Pavičević 12.5, od 17:00 do 18:30, A102@FESB

    Cilj prvog predavanja je da vas uvedemo u svijet izrade web stranica, tj. web programiranja. Predavanje će započeti s kratkim uvodom što je Internet, odnosno na koji način funkcionira. Nakon tih osnova, saznat ćete što su točno web stranice i od čega se sastoje. Saznat ćete što je HTML, a što CSS, kako se pišu, od čega se sastoje i kako se koriste. Sve ćemo popratiti zanimljivim primjerima i praktičnim savjetima koji olakšavaju izradu web stranica. Do kraja predavanja bi trebali naučiti kako napraviti jednostavni statički web. Subotnje predavanje bit će popraćeno radionicom na kojoj bi trebali primijeniti sve naučeno i napraviti svoje prve web stranice. Na radionice ćete se moći prijaviti na predavanjima, stoga nemojte izostati!

  • 2. Praktični CSS i hosting

    Marko Matijević, Frane Roje 19.5, od 17:00 do 19:00, A102@FESB

    Nakon uvodnog predavanja o tome kako radi internet, što je HTML, a što CSS, slijedi drugo predavanje u sklopu te tematike: CSS i hosting stranice u praksi. Naučit ćemo vas kako realizirat demo primjer od prošloga puta, tj. kako složiti HTML i CSS layout po dizajnerskom predlošku. Potom vas učimo i kako objaviti stranicu koju ste napravili, gdje to napraviti i što vam sve treba za objavu. Preporučamo da ne izostanete s ovog predavanja, jer ovoga puta, iznimno, nema radionice u nedjelju!

  • 3. ASP.NET MVC (1. dio)

    Ivan Maček, Frane Roje 26.5, od 17:00 do 19:00, A102@FESB

    Ovim predavanjem završavamo s izradom statičkih web stranica i polako vas uvodimo u svijet web aplikacija. Pokazat ćemo vam kako uz pomoć ASP.NET MVC vaše web stranice mogu postati interaktivne. Ovo je prvo predavanje ciklusa na kojem ćemo programirati. Koristit ćemo C# jezik pa bi trebali znati njegove osnove (više možete doznati koristeći poveznicu video snimaka naše škole C#-a). Nadamo se da ćete ovim predavanjem steći uvid u razvojni ciklus jedne web aplikaciji te se upoznati s osnovnim konceptima ASP.NET MVC-a.

  • 4. ASP.NET MVC (2. dio)

    Ivan Maček, Nebojša Veron 2.6, od 17:00 do 19:00, A102@FESB

    Na četvrtom, ujedno i posljednjem predavanju ovog ciklusa, nastavljamo s programiranjem i razvojem naše web aplikacije koristeći ASP.NET MVC tehnologiju. Naučit ćemo što je Razor view engine, kako dovesti i prikazati podatke na našoj web stranici, te konačno, kako koristeći forme prosljediti podatke web serveru. Ukoliko ste propustili prethodno predavanje, savjetujemo da ga pogledate prije dolaska na četvrto predavanje, jer ćemo koristiti sve naučeno s trećeg predavanja.
